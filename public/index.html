Solar and Wind Plants API

An API aggregating entries of renewable-energy projects, specifically solar and wind energy, all across North America and parts of Mexico. Abridged version of the extensive database



API Endpoints:

  • State
  • Region
  • Net Capacity
  • Plant Name
  • Quarter
  • Fuel Code
  • State
  • Region
  • Power Unit
  • Net Capacity
  • Plant Name
  • Quarter
  • Fuel Code

API Call Example:

By calling an API Endpoint, a list of entries will be returned, or if the provided query doesn't relate to an entry in the database, an empty json object will be returned

Field Name Description
Plant Name The name of the plant
Plant State The state or territory the plant is located in
Unit The kind of technology the plant uses and its type (for example, PV1 is less efficient than PV2, with PV relating to solar photovoltaic cells)
Net Capacity The total capacity of the plant, measured in megawatts
Size Group The size grouping based on the Net Capacity, denoting the electric output of the plant
Current Capacity The current net capacity of the plant, measured in megawatts, same value as the net capcity usually
Fuel Code The type of technology the plant uses to generate energy, SUN is solar and WND is wind energy
Quarter The quarter in a year in which the data is captured from, includes Summer, Fall, Winter, and Spring
Region The region in the United States or the area in North America the plant is located

Get by net capacity

Return a group of projects based on the minimum and maximum query of the net capacity of the plants, in megawatts

Accepts a decimal or whole number

GET http://localhost:8080/netCap?qlower={ minimum value }&qhigher={ maximum value }
